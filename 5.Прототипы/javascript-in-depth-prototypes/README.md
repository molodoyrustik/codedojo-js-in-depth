# JavaScript в деталях - Прототипы

## Установка зависимостей

Перед тем как приступить к выполнению упражнений, необходимо установить программу, которая будет проверять их. Для этого в терминале введите команду

```
npm install
```

## Выполнение упражнений

В каждой из папок имеется 4 файла

* `index.js` - здесь вы будете писать код
* `solution.js` - здесь будет готовое решение, на случай если у вас возникнут проблемы
* `test.js` - здесь находятся тесты, проверящющие ваш код
* `todo.md` - здесь описаны задания, которые вам нужно выполнить в рамках упражнения

## Запуск теста одного упражнения

После того как вы выполнили упражнение, проверьте правильно ли вы все сделали. Для запуска теста одного упражнения введите команду

```
npm test [название папки]
```

Например

```
npm test 01_simple-objects
```

## Запуск тестов всех упражнений

Вы можете проверить сразу все упражнения. Для запуска всех тестов введите команду

```
npm test
```

## Запуск тестов всех упражнений в режиме наблюдения

Чтобы каждый раз не запускать тесты, вы можете запустить их в режиме наблюдения. Тесты будут перепроверятся по мере изменения исходных файлов. Для запуска всех тестов в режиме наблюдения введите команду

```
npm run test:watch
```
